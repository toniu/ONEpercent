# CSV Data Reference

Quick reference for the codes and values used across the CSV files in this folder.

## Ethnicity Codes

| Code | Meaning |
|------|---------|
| WB | White British |
| WI | White Irish |
| WO | White Other |
| IN | Indian |
| PK | Pakistani |
| BD | Bangladeshi |
| CH | Chinese |
| AF | African |
| CB | Caribbean |
| MX | Mixed |

Distribution is loosely based on the 2021 UK census, with adjustments of representation of minority groups for visual diversity in the crowd grid.
